
🎓 Student Result Calculator

A responsive web application that calculates a student's Total Marks, Percentage, Grade, and Pass/Fail Status based on marks entered for five subjects.

✨ Features

✅ User-friendly and responsive interface
✅ Input marks for 5 subjects
✅ Validation for marks between 0 and 100
✅ Automatic Total Marks Calculation
✅ Percentage Calculation
✅ Grade Assignment (A, B, C, D, F)
✅ Pass/Fail Status Display
✅ Instant Results with JavaScript

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

📊 Grading System

Percentage| Grade
90 – 100| A
75 – 89| B
60 – 74| C
40 – 59| D
Below 40| F

🚀 How to Use

1. Open the application in a web browser.
2. Enter marks for all five subjects.
3. Click Calculate Result.
4. View:
   - Total Marks
   - Percentage
   - Grade
   - Pass/Fail Status
