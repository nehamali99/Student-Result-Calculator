🎓 Student Result Calculator

📖 Project Overview

The Student Result Calculator is a responsive web application developed using HTML, CSS, and JavaScript.

This application allows users to enter marks for five subjects and automatically calculates:

- Total Marks
- Percentage
- Grade
- Pass/Fail Status

---

✨ Features

- User-friendly interface
- Responsive design
- Input marks for 5 subjects
- Marks validation (0–100)
- Automatic total marks calculation
- Percentage calculation
- Grade generation
- Pass/Fail status display
- Instant result generation

---

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

---

📊 Grade Criteria

Percentage| Grade
90 - 100| A
75 - 89| B
60 - 74| C
40 - 59| D
Below 40| F

---

🚀 How to Run the Project

1. Download or clone the repository.
2. Open the "index.html" file in any web browser.
3. Enter marks for all five subjects.
4. Click the Calculate Result button.
5. View the calculated result.
